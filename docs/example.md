# RAG 知识库示例文档

这是一个示例文档，用于测试知识库问答功能。

## 什么是 RAG

RAG (Retrieval-Augmented Generation) 是一种结合检索和生成的 AI 技术。
它通过从知识库中检索相关文档，然后将这些文档作为上下文提供给大语言模型，
从而生成更准确、更有依据的回答。

## RAG 的核心流程

1. 用户提出问题
2. 系统将问题转换为向量
3. 在向量数据库中检索最相似的文档片段
4. 将检索到的文档片段作为上下文传递给 LLM
5. LLM 基于上下文生成回答

## 优势

- 减少 LLM 幻觉，回答有据可依
- 知识可以动态更新，无需重新训练模型
- 支持企业私有知识库
